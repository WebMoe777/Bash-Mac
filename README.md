# Bash-Mac

Collection of macOS administration and automation scripts.

## Purpose

This repository contains scripts developed while managing and supporting large-scale Apple environment and tinkering with Apple Devices. 

Topics include:

- Software deployment
- Printer configuration
- WatchGuard installation
- Password management
- Keychain maintenance
